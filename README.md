# Tic Tac Toe Game (React + Vite + Docker)

This is a simple Tic Tac Toe game built with React and Vite, containerized using Docker for easy setup and running.

---

## How to run with Docker

### 1. Make sure Docker is running
Open Docker Desktop and wait until it is active.

---

### 2. Start the project

In the project folder run:

```bash
docker compose up --build
